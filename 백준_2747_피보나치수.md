## MyCode
```python
n = int(input())

def fibo(a):
    if a <=1:
        return a
    else:
        result = fibo(a-1) + fibo(a-2)
        return result

print(fibo(n))
```

## 실패
이 문제는 위의 풀이와 같이 재귀 함수를 사용할 수도 있지만 n이 커질수록 같은 과정을 여러번 거치게 되며(`f(5)를 계산하려면 f(1)이 5번 수행된다`) n이 30이면 약 2^30번(약 10억) 함수를 실행하게 된다. 따라서 `Dynamic Programming` 즉, 동적 계획법을 써야한다. </br>
