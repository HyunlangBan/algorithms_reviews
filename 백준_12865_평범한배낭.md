## MyCode
```python
n, k = map(int, input().split())

array = [0] * (k+1)

### 모든 무게에 대해 최대 가치를 구한다.
for _ in range(n):
    w, v = map(int, input().split())
    array[w] = v
    for i in range(w, k+1):
        data = array[w] + array[i-w]
        if array[i] < data:
            array[i] = data

        print(array)
    
print(array[k])
```
